# Midscene + Remotion Showcase

Video showcase built with [Remotion](https://remotion.dev) and [Midscene.js](https://midscenejs.com) - featuring grid compositions, promo videos, and cool animations.

## Demo

https://github.com/user-attachments/assets/7bd574d7-a457-422e-8ca4-e6f9c64bcb30

## Features

- Four-grid video composition with flying-in animations
- Colored themes and sweep light effects per product
- Center hero video with spring animation and glow pulse
- Midscene.js static cover as first frame with fade-out transition
- Background music with muted video originals

## Getting Started

```bash
cd grid-video
npm install
npm run dev
```

## Render

```bash
cd grid-video
npx remotion render FourGrid out/four-grid.mp4
```
